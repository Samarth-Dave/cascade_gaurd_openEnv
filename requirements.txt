# CRITICAL — torch is pinned to torch==2.6.0+cu124:
#   • HF Space GPU nodes randomly have CUDA 12.9 OR 13.0 drivers.
#   • Default `pip install torch` gets 2.11+cu130 → fails on CUDA 12.9 nodes.
#   • cu124 wheels work on ANY driver >= 12.4 (covers both 12.9 and 13.0).
#   • Python 3.13 (the Space runtime) needs torch >= 2.6.0.
#   • cu121 wheels stopped at 2.5.1 (no py313 build), so cu124 is the only
#     option that satisfies BOTH "supports py313" AND "works on driver 12.9".
# ─────────────────────────────────────────────────────────────────────────────
