# CascadeGuard Result Assets

Drop final judge-facing plots and screenshots in this folder. Keep filenames stable so README links work without edits.

Recommended files:

| File | Purpose |
|---|---|
| `training_loss_curve.png` | SFT/GRPO loss over training step |
| `training_reward_curve.png` | GRPO reward over training step |
| `baseline_vs_trained_reward.png` | Random or heuristic baseline vs trained agent |
| `hospital_uptime_before_after.png` | Critical-service protection before/after training |
| `demo_dashboard.png` | Screenshot of the live HF Space dashboard |
| `episode_trace_before_after.png` | Qualitative action trace comparison |

Do not commit large videos here. Link videos, blog posts, or slides from the README.
